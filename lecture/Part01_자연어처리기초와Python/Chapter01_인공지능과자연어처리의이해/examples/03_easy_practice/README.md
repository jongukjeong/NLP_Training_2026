# 오늘의 쉬운 실습: 규칙 기반 문의 분류

함수, 딕셔너리, 점수 계산 없이 `if`, `elif`, `for`만 사용합니다.

## 진행 순서

1. `01_one_question.py`: 질문 하나에서 `배송` 찾기
2. `02_two_intents.py`: 환불과 배송 구분하기
3. `03_many_questions.py`: 질문 여러 개 분류하기
4. `04_basic_practice.py`: 분류 결과에 답변 연결하기

```powershell
cd examples\03_easy_practice
python 01_one_question.py
python 02_two_intents.py
python 03_many_questions.py
python 04_basic_practice.py
```

각 파일을 실행한 뒤 키워드나 질문을 하나 바꿔 결과 변화를 확인합니다.

## 오늘의 완료 기준

- `if` 조건에서 찾는 키워드를 말할 수 있다.
- 질문 하나를 추가할 수 있다.
- 새로운 문의 유형 하나를 `elif`로 추가할 수 있다.
- 키워드가 없을 때 기타 문의가 되는 이유를 설명할 수 있다.

점수 계산, 딕셔너리 데이터 분리와 함수화는 미니 프로젝트 solution에서 비교합니다.
