# Chapter 1 미니 프로젝트 Starter

오늘의 쉬운 실습을 마친 뒤 시작하는 기본 프로젝트입니다.

## 1단계: 제공 코드 실행

```powershell
cd projects\starter
python faq_chatbot_starter.py
```

`배송이 아직 안 왔어요.`를 입력해 결과를 확인합니다.

## 2단계: 질문 바꿔 보기

`test_questions.txt`의 질문을 하나씩 입력하고 분류 결과를 기록합니다.

## 3단계: 문의 유형 추가

다음 중 하나만 선택해 `elif` 조건과 답변을 추가합니다.

- 결제 문의
- 계정 문의
- 상품 문의

## 기본 완료 기준

- 프로그램을 실행하고 질문을 입력했다.
- 새로운 문의 유형을 하나 추가했다.
- 성공 질문 2개와 실패 질문 1개를 찾았다.
- 키워드 방식의 한계를 한 문장으로 설명했다.

함수, 딕셔너리, 점수 계산, 반복 입력과 자동 테스트는 기본 완료 기준이 아닙니다. 수강생의 기본 구현을 확인한 뒤 [완성형 solution](../README.md)을 공개합니다.
