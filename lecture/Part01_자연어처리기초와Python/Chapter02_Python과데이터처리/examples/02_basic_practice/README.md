# Basic Practice: 기본 데이터 정제

Step by Step에서 배운 처리를 하나의 짧은 프로그램으로 합칩니다.

먼저 `basic_data_cleaning.py`의 빈칸 없이 전체 흐름을 읽고, 다음 순서를 말로 설명해 봅니다.

```text
CSV 읽기 → 결측 제거 → 공백 정리 → 레이블 통일 → 중복 제거 → 저장
```

실행:

```powershell
python basic_data_cleaning.py
```

확인할 내용:

- 처리 전 6행이 처리 후 3행이 된 이유
- `Positive`와 `NEGATIVE`가 어떻게 바뀌었는지
- `reviews_clean.csv`에 어떤 열이 추가됐는지
