# Step by Step: 한 단계씩 따라 하기

강사와 함께 실행 결과를 확인하는 입문 예제입니다. 각 파일은 앞 단계의 코드를 포함하고 한 가지 처리만 추가합니다.

1. `01_read_csv.py`: CSV 읽기와 구조 확인
2. `02_remove_missing.py`: 결측 문장 제거
3. `03_clean_text.py`: 공백 정리와 빈 문장 제거
4. `04_clean_label.py`: 레이블 표기 통일
5. `05_remove_duplicates.py`: 중복 문장 제거
6. `06_save_result.py`: 결과 저장과 행 수 비교

이 폴더에서 순서대로 실행합니다.

```powershell
python 01_read_csv.py
python 02_remove_missing.py
python 03_clean_text.py
python 04_clean_label.py
python 05_remove_duplicates.py
python 06_save_result.py
```

각 단계에서 `print()` 결과가 어떻게 바뀌었는지 말로 설명한 뒤 다음 단계로 이동합니다.
