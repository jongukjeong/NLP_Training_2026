# 미니 프로젝트 시작 코드

`text_data_explorer_starter.py`의 1~10단계를 순서대로 완성합니다. 처음에는 함수, CLI 인수, JSON 보고서를 구현하지 않습니다.

진행 방법:

1. 2~4단계까지 작성하고 실행합니다.
2. 중간 결과를 `print(df)`로 확인합니다.
3. 5~7단계를 작성하고 다시 실행합니다.
4. 8~10단계를 완성합니다.
5. 결과를 설명할 수 있으면 선택 도전에 참여합니다.

```powershell
python text_data_explorer_starter.py
```

완성형 solution은 기본 요구사항을 직접 시도한 뒤 강사의 안내에 따라 확인합니다.
