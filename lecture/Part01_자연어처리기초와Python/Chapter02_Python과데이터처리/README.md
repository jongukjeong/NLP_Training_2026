# Chapter 2. Python과 데이터 처리

자연어 처리 코드는 결국 **문자열을 읽고, 구조화하고, 정제하고, 다시 저장하는 Python 프로그램**입니다. 이 장에서는 문법을 암기하는 대신 실제 텍스트 데이터를 다루는 흐름 안에서 Python의 핵심 기능을 익힙니다.

## 학습 목표

이 장을 마치면 다음을 할 수 있습니다.

- 변수와 기본 자료형을 설명하고 적절히 선택한다.
- 문자열, 리스트, 튜플, 집합, 딕셔너리로 텍스트 데이터를 표현한다.
- 조건문, 반복문, 함수로 재사용 가능한 처리 절차를 만든다.
- 텍스트, CSV, JSON 파일을 UTF-8로 안전하게 읽고 쓴다.
- pandas로 표 데이터를 탐색하고 결측값·중복·문자열을 정제한다.
- 예외 처리와 검증을 포함한 작은 데이터 처리 프로그램을 완성한다.

## 예제 파일 배치 규칙

각 예제의 Python 코드와 그 코드가 사용하는 데이터셋은 **동일한 예제 폴더**에 둡니다.

```text
examples/chapter02/text_data_explorer/
├── text_data_explorer.py
└── reviews.csv
```

실행 결과는 원본과 섞이지 않도록 같은 폴더 아래 `output/`에 저장합니다.

> 예제는 Windows PowerShell 기준 명령도 함께 제시하지만, Python 코드는 운영체제와 무관하게 실행되도록 작성합니다.
