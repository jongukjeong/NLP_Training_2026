# Chapter 3 전처리 실험과 오류 분석

## 전처리 규칙을 실험으로 결정하기

전처리 규칙은 “깔끔해 보인다”가 아니라 최종 과제 성능과 오류 유형으로 선택합니다. 규칙 A와 B를 같은 데이터 분할에서 비교하고 한 번에 한 규칙만 바꿉니다.

| 실험 | 변경 규칙 | 빈 문장률 | 미등록어율 | Val F1 | 결론 |
|---|---|---:|---:|---:|---|
| base | 공백만 통일 | 기록 | 기록 | 기록 | 기준 |
| exp01 | 특수문자 제거 | 기록 | 기록 | 기록 | 비교 |
| exp02 | 반복문자 2회 제한 | 기록 | 기록 | 기록 | 비교 |

## 처리 함수의 입출력 계약

```python
def preprocess(text: str) -> str:
    """문자열 하나를 받아 정제 문자열 하나를 반환한다."""
    if text is None:
        return ""
    text = str(text)
    text = normalize_unicode(text)
    text = normalize_spaces(text)
    return text.strip()
```

함수는 같은 입력에 같은 결과를 내야 하며 학습과 서비스에서 공유합니다. 데이터프레임 전체에 의존하는 통계 처리는 별도 `fit/transform` 단계로 구분합니다.

## 규칙별 단위 테스트

```python
cases = [
    ("배송이  늦어요", "배송이 늦어요"),
    ("  환불 요청  ", "환불 요청"),
    ("A-104 오류", "A-104 오류"),
]
for raw, expected in cases:
    assert preprocess(raw) == expected
```

대표 사례뿐 아니라 빈 문자열, `None`, 숫자만 있는 문장, 이모지, 매우 긴 문장을 포함합니다.

## 과도한 정제 찾기

원문과 결과 길이 비율을 계산합니다.

\[
보존율=\frac{전처리후문자수}{원본문자수}
\]

보존율이 0.1인 문장은 원문의 90%가 사라졌다는 뜻입니다. 낮은 보존율 사례를 정렬해 규칙을 검토합니다.

```python
ratio = len(cleaned) / max(len(raw), 1)
```

## 부정어 보존 검사

`안`, `못`, `없다`, `아니다`는 감성을 바꿉니다. 전처리 후 이 표현이 사라진 문장을 자동으로 찾습니다.

```python
negations = ["안", "못", "없", "아니"]
if any(n in raw for n in negations) and not any(n in clean for n in negations):
    print("부정 정보 손실", raw, clean)
```

## 숫자와 코드

검색·고객지원에서는 `A-104`, `5G`, `100GB`, 날짜와 금액이 핵심일 수 있습니다. 숫자를 모두 `<NUM>`으로 바꾸는 실험과 원문 유지 실험을 분리합니다.

## 형태소 분석 결과 검토

형태소 분석기는 정답 생성기가 아닙니다. 도메인 신조어와 제품명을 잘못 나눌 수 있습니다.

```text
원문: 챗GPT플러스 결제 오류
도구 A: 챗 / GPT / 플러스 / 결제 / 오류
도구 B: 챗GPT플러스 / 결제 / 오류
```

최종 과제에서 어떤 결과가 더 유용한지 판단하고 사용자 사전 적용 여부를 비교합니다.

## Tokenizer 변경의 영향

Tokenizer를 바꾸면 vocabulary, sequence length, OOV와 모델 입력이 모두 변합니다. 이전 모델 가중치나 vectorizer와 섞어 쓰지 않습니다.

## 데이터 분할 시점

중복 제거와 그룹 분할을 먼저 계획합니다. 전체 데이터 빈도를 보고 불용어를 선택하면 test 정보가 들어갈 수 있습니다. 학습 데이터로 규칙을 결정하고 test는 마지막 확인에 사용합니다.

## 오류 샘플링

무작위 50건 외에 다음 위험 그룹을 각각 20건씩 확인합니다.

- 가장 긴 문장
- 가장 많이 변경된 문장
- 숫자·URL·이모지 포함
- 부정어 포함
- 처리 후 빈 문자열
- 희귀 문자와 외국어 포함
