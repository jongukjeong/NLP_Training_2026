# 04. 토큰화

토큰화는 텍스트를 모델이나 통계가 처리할 단위로 나누는 과정입니다. 하나의 정답이 아니라 목적과 모델에 따른 선택입니다.

## 토큰 단위 비교

| 단위 | 예 | 장점 | 한계 |
|---|---|---|---|
| 문자 | `자`, `연`, `어` | 미등록어 없음 | 시퀀스가 길어짐 |
| 어절 | `자연어`, `처리는` | 단순·빠름 | 조사·어미가 결합됨 |
| 형태소 | `자연어`, `처리`, `는` | 한국어 구조 반영 | 분석기·사전 의존 |
| 서브워드 | 모델별 조각 | 미등록어와 어휘 크기 균형 | 모델 토크나이저 필요 |

## 문장 분리

마침표만으로 분리하면 URL, 소수, 약어를 잘못 나눌 수 있습니다. 단순 예제는 한계를 명시하고 실제 시스템에서는 언어별 문장 분리기를 검토합니다.

```python
import re

def split_sentences(text: str) -> list[str]:
    parts = re.split(r"(?<=[.!?])\s+|[\r\n]+", text)
    return [part.strip() for part in parts if part.strip()]
```

## 어절 토큰화

```python
def whitespace_tokenize(text: str) -> list[str]:
    return text.split()
```

빠른 기준선에는 유용하지만 `좋다`, `좋아요`, `좋았습니다`를 서로 다른 토큰으로 봅니다.

## 모델 토크나이저

BERT나 GPT 계열 모델은 모델과 함께 제공되는 토크나이저를 사용해야 합니다. 같은 문장도 모델마다 토큰 ID가 다르며, 임의 토큰화 결과를 모델 입력에 사용하면 안 됩니다.

## 품질 지표

- 문장당 평균 토큰 수
- 빈 토큰 결과 비율
- 지나치게 긴 문장 비율
- 고유 토큰 수와 저빈도 토큰 비율
- 처리 전후 표본 비교

## 재현성

토크나이저 이름, 버전, 설정, 사용자 사전을 함께 기록합니다. 모델 토크나이저는 모델 리비전도 고정하는 것이 좋습니다.
