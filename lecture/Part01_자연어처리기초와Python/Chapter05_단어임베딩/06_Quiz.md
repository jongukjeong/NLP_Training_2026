# 06. 퀴즈

## 문제

1. 임베딩이 BoW보다 밀집 표현이라고 불리는 이유는 무엇인가요?
2. 분포 가설은 무엇인가요?
3. 문맥 창 크기가 임베딩에 미치는 영향은 무엇인가요?
4. CBOW와 Skip-gram의 예측 방향 차이는 무엇인가요?
5. OOV란 무엇인가요?
6. 내재 평가만으로 충분하지 않은 이유는 무엇인가요?
7. 임베딩 유사도를 사실 관계로 해석하면 안 되는 이유는 무엇인가요?

## 정답

1. 비교적 적은 차원의 대부분에 실숫값을 저장하기 때문입니다.
2. 비슷한 문맥에 등장하는 단어는 의미도 비슷하다는 가정입니다.
3. 작은 창은 국소 관계, 큰 창은 주제 관계를 더 반영하는 경향이 있습니다.
4. CBOW는 문맥에서 중심 단어를, Skip-gram은 중심 단어에서 문맥을 예측합니다.
5. 학습 어휘에 없는 단어입니다.
6. 그럴듯한 유사도가 실제 과제 성능을 보장하지 않기 때문입니다.
7. 학습 데이터의 빈도와 사회적 편향이 벡터에 반영되기 때문입니다.
