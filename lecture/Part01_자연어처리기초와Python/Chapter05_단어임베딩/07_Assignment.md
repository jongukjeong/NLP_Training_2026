# 07. 실습 과제

## 과제: 작은 동시출현 임베딩

`sentences.txt`를 사용해 다음을 수행합니다.

1. 공백 토큰으로 어휘를 만듭니다.
2. 창 크기 2의 동시출현 행렬을 만듭니다.
3. SVD로 밀집 벡터를 생성합니다.
4. `배송`과 가장 가까운 단어 5개를 구합니다.
5. 어휘 크기, 차원, 검색 결과를 JSON으로 저장합니다.

작은 교육용 말뭉치의 결과를 일반적인 언어 지식으로 해석하지 않습니다.
