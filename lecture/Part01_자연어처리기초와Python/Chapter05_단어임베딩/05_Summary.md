# 05. 핵심 정리

```text
토큰 말뭉치 → 어휘·문맥 정의 → 동시출현 또는 예측 학습
→ 밀집 벡터 → 유사도·다운스트림 평가 → 버전 관리
```

- 임베딩은 단어를 저차원 밀집 벡터로 표현한다.
- 비슷한 문맥에 등장하는 단어가 가까워지는 분포 가설을 따른다.
- 문맥 창은 임베딩이 학습하는 관계에 영향을 준다.
- SVD는 동시출현 행렬을 저차원으로 축소할 수 있다.
- CBOW는 문맥에서 중심 단어를, Skip-gram은 중심 단어에서 문맥을 예측한다.
- OOV, 작은 말뭉치, 다의어와 데이터 편향을 확인해야 한다.
- 실제 목적에 가까운 외재 평가가 중요하다.
