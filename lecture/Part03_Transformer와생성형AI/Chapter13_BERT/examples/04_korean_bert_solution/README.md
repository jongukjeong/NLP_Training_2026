## 공개 시점과 사용 방법

이 자료는 수강생이 기본 실습을 먼저 시도하고 피드백을 받은 뒤 공개하는 완성형 참고 자료입니다. 기본 코드보다 복잡한 것이 정상이며, 전체를 복사하기보다 자신의 코드와 구조·검증·오류 처리 방식을 비교합니다.

```powershell
python korean_bert_embeddings.py
```

기본 모델은 `klue/bert-base`이며 `HF_BERT_MODEL`로 교체할 수 있습니다. 첫 실행에는 모델 다운로드가 필요합니다. 모델 ID와 결과 차원을 `output/report.json`에 기록합니다.
